# Contessa Shop

Contessa Shop is a focused Next.js storefront concept for a three-piece essentials drop. The homepage pairs a Skims-inspired product grid with item selection, size/color controls, rotating product angles, and a PCRF impact section.

## Features

- Current Drop product grid with three selectable items.
- Per-item color and size selection.
- Product-angle carousel with automatic rotation and side arrow controls.
- PCRF section with background imagery, transparent logo treatment, and an outbound PCRF link.
- Responsive desktop and mobile layouts.

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS
- Lucide React icons

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the local development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```

## Environment

No environment variables are required for the storefront demo.
